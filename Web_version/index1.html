Welcome to Telegram Theme Maker

Our tool allows you to easily create a theme for telegram Desktop

The first Theme Maker for Telegram.

This tool is the first theme Maker for Telgram Desktop.
We choosed to adopt a differnt way of creatting themes which allow every user wich have an acces to the internet (or not, got to github to discover our desktop version).
There are two versions wich you can use to create your theme.
The first is a Fast (also called simple over internet), wich allows you to choose the manin colors, a remote jpg image (other formats will corupt your theme, sorry for this) and a name. You can then download it in two versions. Just colors or colors + image. Send them to the chat with yourself and apply it.
